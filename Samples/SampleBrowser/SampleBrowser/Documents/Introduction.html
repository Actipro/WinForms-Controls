Thank you for downloading the Actipro WinForms Controls set of user interface controls. These components allow you to implement advanced interaction functionality in your Windows Forms applications. Learn more about our products (SyntaxEditor, Bars, Docking/MDI, Navigation, and Wizard) below, or navigate through product information and samples on the left.

Our WinForms Controls NuGet packages support .NET, .NET Core, and .NET Framework. The Sample Browser project included with our installer supports Visual Studio 2017 and later, and targets .NET Framework 4.7.2. A multi-targeted NuGet-based Sample Browser project can also be downloaded from GitHub.

SyntaxEditor is a powerful text editing control that is packed with features for efficient code editing, just like the Visual Studio code editor. Features include syntax highlighting, code outlining, parsing, line numbers, block selection, IntelliPrompt UI (completion list, parameter info, quick info, code snippets, smart tags), split views, single-line mode, and much more. Language definitions can be developed for any custom language.

Several premium language add-ons for C#, VB, XML, JavaScript, JSON, and Python are also available that enhance the SyntaxEditor code-editing experience with features like parsing, syntax error reporting, automated code outlining, and IntelliPrompt.

Bars consists of docking toolbars, menus, and status bars. The entire system is based on a command-driven design that allows customizable command links to be dropped anywhere throughout the dockable toolbar and menu layout. Popup/split buttons, chord shortcuts, tear-off menus, MDI merged menus, and more are supported.

Docking/MDI includes powerful docking window management functionality, Tool windows can be dragged and docked on any side of the host container, nested in hierarchies, or floated in their own rafted window. Additionally they can be collapsed into auto-hide mode and hidden, or moved to the MDI area. Tabbed and standard MDI (multiple document interface) modes are both supported, along with tool window inner-fill mode.

Navigation contains a NavigationBar control that facilitates navigating through different panes of data. It has an elegant user interface that uses buttons to control which navigation pane is selected. Panes can display data or simply act as a container for child controls. If screen real estate is available, large buttons can be used for pane selection. The end-user can use a splitter to move the large buttons into the overflow button tray, which uses smaller buttons.

Wizard dialogs simplify the user experience of applications by taking complex tasks and breaking them up into a series of simple steps. Actipro Wizard is a lightweight control for building these wizard dialogs, while conforming to the Wizard97 specifications.
